
## Frontend

### Folder Structure

All the JS files are inside the src folder.
```
|_ src 
  |_components
  |_css
  |_hoc
  |_layout
  |_pages
  |_app.jsx
  |_index.jsx
```
 Pages folder contain the code for all web pages.


## How to run React App ?

1.Navigate to ReactFrontend folder

```bash
  cd ReactFrontend
```
2.Install required packages

```bash
  npm install
```
3. Start the application
```bash
npm start
```
